Welcome to the Java FM Playground!

The Java Foundation Model Playground is a sample application showcasing how to leverage Amazon Bedrock from Java code. As any sample application, it is not production-ready. It is provided for the sole purpose of illustrating how Java developers can leverage Amazon Bedrock to build generative AI-enabled applications.
Amazon Bedrock is a fully managed service that offers a choice of high-performing foundation models (FMs) from leading AI companies like AI21 Labs, Anthropic, Cohere, Meta, Stability AI, and Amazon via a single API.
